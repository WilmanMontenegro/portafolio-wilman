01 · LLM

02 · Prompts

Prompt = lo que le pides a la IA

La instrucción que escribes en el chat. Un buen prompt lleva 4 piezas:
01 Rol

Quién es la IA

02 Tarea

Qué debe hacer

03 Contexto

Marca, tono, límites

04 Formato

Largo, estructura, cómo termina

Sin estas 4, la IA improvisa… y tú pierdes tiempo. Hazlos con tu marca.

Ejemplo real · Instagram

Malo vs bueno

Prompt malo

Escribe un copy para Instagram.

Vago. Sin marca, tono ni formato. Respuesta genérica.

Prompt bueno

Eres copywriter de una cafetería en Barranquilla. Escribe 1 caption de Instagram (máx. 80 palabras) para lanzar un café frío. Tono: cercano y costeño, sin slang exagerado. Incluye: 1 beneficio, 1 CTA suave, 3 hashtags locales. No inventes precios ni premios.

03 · Token y ahorrar

Token = pedacito de texto

Se cobra por pedacitos, no por “palabras” · Entrada barata · Salida cara
Regla práctica

1 token ≈ 3–4 caracteres

100 palabras ≈ 130–150 tokens

Caption corto ≈ 40–80 · post largo ≈ 200–400

Por qué te importa

Cada mensaje reenvía el chat: lo anterior + lo nuevo + la respuesta.

No “recuerda” gratis. Chat largo = más gasto. Sé claro y corto.

Entrada

Lo que tú escribes.
Más barato.

Salida

Lo que responde la IA.
3× a 6× más caro.

Pide formato corto. No pegues chats viejos.

04 · Ventana de contexto

Contexto = lo que puede leer de una vez

Tu pregunta + el chat anterior + archivos. Tiene tope. No es memoria eterna.
Chat nuevo por tarea

Un chat para captions. Otro para el correo. No mezcles temas.

Solo lo necesario

Resumen, tono, precio, público. No pegues todo el Drive “por si acaso”.

Si el chat se alarga

Resume lo importante… o chat nuevo.

Archivos con criterio

Sube el PDF o documento correcto. Menos archivos = menos ruido.

Buen contexto = lo justo para la tarea. No es “más = mejor”.

Ahorrar bien

El modelo correcto

En ChatGPT, Claude o Gemini hay varios modelos. No uses el más Pro para todo.
Barato · rápido

Resumir, clasificar, borradores simples

→
Fuerte · caro

Estrategia y decisiones difíciles

Barato primero. Caro solo si hace falta. Eso es tu retorno: menos gasto y más tiempo.

Ejemplos reales

Las 3 apps más usadas

ChatGPT

Gratis → modelo automático (en el chat normal casi no eliges)

Plus → puedes subir el “razonamiento” si la tarea es difícil

Familia GPT-5.6 → Luna / Terra / Sol (más en plan pago o Work)

Idea: empieza simple. Sube solo si no alcanza.

Claude

Haiku 4.5 → clasificar contactos, respuestas cortas

Sonnet 5 → posts, correos y anuncios

Opus 4.8 → estrategia o análisis difícil

Extra: Fable 5 → lo más difícil (solo si Opus no alcanza)

Gemini

3.1 Flash-Lite → borradores rápidos y mucho volumen

3.5 Flash → resumir Docs/Sheets o chats del día a día

3.1 Pro → auditoría o tareas difíciles (math, código, varios archivos)

Extra: Razonamiento ampliado → problemas complejos

Empieza en el barato. Sube solo si la respuesta no alcanza.

05 · Riesgos

Tres riesgos reales

Alucinaciones

Inventa cosas que suenan verdad.

Sesgos

Puede repetir prejuicios.

Privacidad

No pegues datos sensibles (contraseñas, clientes, docs privados).

Prompt de internet: léelo. Si pide datos o “ignora reglas”, no lo uses.

La IA propone. Tú verificas. Mejor prompts con tu marca.

Prompt listo para copiar

Presenta tu producto

Izquierda: edita los [corchetes]. Derecha: ejemplo listo.
Plantilla · edítala

Crea una presentación de [8] diapositivas para el producto [nombre del producto] de la marca [nombre de la marca]. Negocio: [a qué se dedica]. Público: [cliente ideal]. Objetivo: [vender / captar clientes / capacitar]. Ciudad o zona: [ciudad]. Tono: [cercano / profesional]. Idioma: español. Estructura: 1) Portada con promesa 2) Problema del cliente 3) Solución / producto 4) 3 beneficios claros 5) Cómo funciona o qué incluye 6) Precio o forma de comprar: [precio o “consultar”] 7) Prueba social: [testimonio o caso] 8) CTA: [WhatsApp / web / agendar] Poco texto por slide. No inventes cifras.

Ejemplo · cópialo tal cual

Crea una presentación de 8 diapositivas para el producto “Kit de café en grano” de la marca Café Costa. Negocio: cafetería y venta de café tostado en Barranquilla. Público: oficinas y personas que trabajan desde casa en la Costa. Objetivo: vender el kit por WhatsApp. Ciudad o zona: Barranquilla, Colombia. Tono: cercano y cálido. Idioma: español. Estructura: 1) Portada con promesa 2) Problema del cliente 3) Solución / producto 4) 3 beneficios claros 5) Cómo funciona o qué incluye 6) Precio: desde $45.000 7) Prueba social: pedidos semanales por WhatsApp 8) CTA: WhatsApp para pedir Poco texto por slide. No inventes cifras.

Pega en Gamma. Usa la plantilla con tu negocio.
