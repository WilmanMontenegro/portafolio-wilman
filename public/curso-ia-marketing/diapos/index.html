01

¿Qué es un LLM?

LLM = Large Language Model
(modelo de lenguaje grande)

Concepto base

Modelo de lenguaje grande

Large Language Model · LLM
En simple

Una IA que entiende y genera texto.

Cómo funciona

Predice la siguiente palabra.

Idea clave

No es una base de datos

No “busca y encuentra”. Completa el texto.
patrones + tu pregunta
→ siguiente palabra
Puede sonar bien… y estar mal.

En marketing digital

No solo texto: también imagen, video, datos y atención al cliente

La IA acelera. Tú decides marca, verdad y estrategia.

02

¿Qué es un token?

Token = pedacito de texto
Así mide y cobra la IA

Unidad básica

Token = pedacito de texto

Se cobra por tokens, no por palabras
Regla OpenAI (solo inglés)

1 token ≈ 4 caracteres

1.000 tokens ≈ 750 palabras

Fuente: OpenAI Help Center

En español

Mismo mensaje ≈ más tokens (~20–50% más)

1.000 tokens ≈ 550–650 palabras

Medidas con tiktoken (GPT-4 / GPT-4o)

03

¿Qué es el contexto?

Lo que la IA “ve” en este momento
Se mide en tokens

Ventana de contexto

Cuánto puede leer de una vez

No es memoria eterna. Es lo que cabe ahora.
Qué entra

Tu pregunta + el chat anterior + archivos que subes

El límite

Cada modelo tiene un tope de tokens.
Si te pasas, “olvida” lo de arriba.

Más contexto útil ≠ mejor respuesta. Elige qué meter.

Cómo se cobra

Pagas por cada pedacito

Entrada = lo que metes al contexto · Salida = la respuesta
Entrada

Lo que tú escribes.
Más barato.

Salida

Lo que responde la IA.
3× a 6× más caro.

Ahorrar bien

El modelo correcto

Misma casa, varios modelos. No uses el más caro para todo.
Barato · rápido

Resumir, clasificar, borradores simples

→
Fuerte · caro

Prompts prácticos

Ejemplos para tu negocio
Malo vs bueno · listos para copiar

Receta corta

Cuatro piezas de un buen prompt

01 Rol

Quién es la IA

02 Tarea

Qué debe hacer

03 Contexto

Marca, tono, límites

04 Formato

Largo, estructura, CTA

Sin estas 4, la IA improvisa… y tú pierdes tiempo.

Ejemplo real · Instagram

Malo vs bueno

Prompt malo

Escribe un copy para Instagram.

Vago. Sin marca, tono ni formato. Respuesta genérica.

Prompt bueno

Eres copywriter de una cafetería en Barranquilla. Escribe 1 caption de Instagram (máx. 80 palabras) para lanzar un café frío. Tono: cercano y costeño, sin slang exagerado. Incluye: 1 beneficio, 1 CTA suave, 3 hashtags locales. No inventes precios ni premios.

Copia el bueno. Cámbialo a tu negocio. Pruébalo hoy.

09

Retorno de inversión

ROI = beneficio vs lo que gastas

Estrategia

Barato primero.
Caro solo si hace falta.

Modelo barato → filtrar
Modelo caro → decidir

Impacto

26–75%

menos tiempo en tareas si usas bien la IA

Para llevar

Recuerda esto

→La IA predice palabras. No “sabe” todo.
→Token mide y cobra. Contexto = lo que cabe ahora.
→Prompt claro: rol + tarea + contexto + formato.
→Decks rápidos: NotebookLM, Claude Design (o Gamma).
→La respuesta cuesta más que la pregunta.
→Siempre revisa antes de publicar.
→Modelo barato primero. Caro con intención.
